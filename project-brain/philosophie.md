# Philosophie éditoriale de WiggMap

Le ton de WiggMap doit toujours rester humain, naturel et agréable à lire.

Le site ne doit jamais donner l'impression d'être écrit par une machine ou par un outil informatique.

Le style attendu est :

- fluide
- bien écrit
- vivant
- attractif
- clair
- crédible
- accessible

Le contenu doit être informatif sans être froid.

Il faut éviter :

- le ton robotique
- les phrases trop génériques
- les tournures creuses
- le jargon technique inutile
- l'écriture trop scolaire

Le contenu doit donner envie de lire, même quand il parle de données, de salaires, de loyers, de fiscalité ou de qualité de vie.

Les données doivent être présentées de manière simple, lisible et intéressante.

WiggMap doit mélanger :

- information utile
- curiosité
- comparaison
- découverte
- culture générale
- projection de vie

Même quand le site aide à comparer des pays de manière rationnelle, il doit garder une dimension humaine et inspirante.
